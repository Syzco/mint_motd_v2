Installation-
1. Unzip.
2. Drag and drop the mintmotd addon into your addons directory.
3. Edit your config. 
3. a. Config can be found in lua/config. 
3. b. To enable/disable packages use config.lua
3. c. To modify maturity, use maturity.lua
3. d. To modify motd, use motd.lua
4. Ready to use!

-----------------------------------------------------------------------------------------------------------
All config is located under sh_bm_config.lua. 
This can be found at mintmotd/lua/autorun/sh_bm_config.lua.
The configs have comments in there to help guide you through them.
